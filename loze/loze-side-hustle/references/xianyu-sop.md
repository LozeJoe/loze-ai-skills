# 接单交付 SOP — Loze + Hermes Agent 协作

## 核心分工

| 角色 | 职责 |
|------|------|
| Loze | 闲鱼接单、沟通需求、报价、收款、交付 |
| Hermes（我） | 写代码、调试、生成交付物 |

---

## 标准流程

### Step 1：接单沟通

买家私聊你 → 问清楚需求：
1. 「具体要做什么？」（功能/效果/输入输出）
2. 「什么时候要？」（评估时间是否来得及）
3. 「有参考图/示例吗？」

然后你把需求发给我（微信就行），我评估可行性 + 报价建议。

### Step 2：报价

参考价格表：
| 类型 | 起步价 | 典型范围 |
|------|--------|----------|
| 简单脚本（<50行） | ¥50 | 50-100 |
| 中等脚本（50-200行） | ¥100 | 100-200 |
| 复杂项目 | ¥200 | 200-500+ |
| AI工具安装 | ¥30 | 30-80 |
| 网页修改 | ¥100 | 100-300 |
| 整页/组件 | ¥200 | 200-800 |

报价原则：比市场价低 20-30%（学生身份 + AI 效率优势）

### Step 3：我写代码

你把需求发我 → 我直接写代码 → 返回给你：
- 完整可运行代码
- 使用说明（如果有）
- 可能的交付物（.py / .vue / .java / .zip）

### Step 4：你交付

- 把文件发给买家
- 简单指导怎么用
- 确认买家满意 → 点「已发货」→ 等确认收货

### Step 5：售后

小修改免费（≤15分钟工作量），大改动另报价。

## 注意事项

### ✅ 可以做
- Python脚本、Excel处理、自动化
- AI工具安装配置
- 网页/Vue前端开发
- Java后端接口/Bug修复
- 合法公开数据采集
- 技术指导/代码审查

### ❌ 不做（触反诈红线 A6）
- 代写完整毕设/课程设计
- 考试作弊/代考
- 破解/绕过网站安全防护
- 刷量/刷单/灰产
- 采集个人隐私信息

### 💰 交易安全
- 必须走闲鱼担保交易，不私下转账
- 不先交付完整代码再等付款
- 复杂项目可以分阶段交付（先做核心功能 → 确认满意 → 继续 → 最后交付）
